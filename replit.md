# Hamza Ahmed Portfolio Website

## Overview
This is a personal portfolio website for Hamza Ahmed, a front-end web developer with over 4 years of experience. The portfolio showcases skills, projects, testimonials, and contact information.

## Project Structure
- **Main Portfolio**: `index.html`, `style.css`, `script.js` - Main portfolio landing page
- **Sub-projects**: Multiple interactive web applications in subdirectories:
  - `calculator/` - A calculator application
  - `Chess/` - Chess game
  - `cube/` - 3D cube visualization
  - `login/` - Login interface
  - `music-player/` - Music player application
  - `solar/` - Solar system visualization
  - `tab-bar/` - Tab bar component
  - `tower-blocks/` - Tower blocks game
  - `scroll/` - Scroll effect demonstration

## Technology Stack
- Pure HTML5, CSS3, and JavaScript (Vanilla JS)
- No build tools or frameworks required
- Static website deployment

## Setup & Deployment
- **Development**: Simple Python HTTP server on port 5000
- **Server**: `server.py` serves static files with proper cache control headers
- **Host**: Configured to bind to 0.0.0.0:5000 for Replit environment
