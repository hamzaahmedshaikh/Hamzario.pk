# evelyn-portfolio

Production-ready MERN portfolio for Evelyn Grace Carter (Full-Stack MERN Developer).

## Quick Start
1. `git clone <repo-url>`
2. `cd evelyn-portfolio && npm run bootstrap`
3. Set up `.env` (copy from `.env.example`)
4. `npm run seed` or `cd backend && npm run seed`
5. Backend: `cd backend && npm run dev`
6. Frontend: `cd frontend && npm run dev`

Admin login: Firebase Auth (see Admin section for credentials).
